# 实验报告待补充内容说明

## 需要补充的两个内容（在已有的大纲内插入）

### 1. 超参数调整过程

**建议内容**（800-1000字，段落式专业论述）：
- 初始超参数的选择依据（如为什么选择学习率0.1、批次大小128等）
- 调整过程中的关键发现：
  * 学习率的影响（测试过哪些值，如0.01, 0.05, 0.1, 0.2等，观察到的现象）
  * 批次大小的影响（如64 vs 128 vs 256的对比）
  * 权重衰减系数的影响（1e-3, 5e-4, 1e-4等的对比）
  * 标签平滑系数的影响（0, 0.05, 0.1, 0.15等的对比）
  * Cutout遮挡大小的影响（8x8, 12x12, 16x16, 20x20等的对比）
- 不同超参数组合的实验结果对比（可以列举2-3组典型的配置及其结果）
- 最终超参数配置的确定过程（如何在多次实验中逐步优化到最佳配置）
- 调参过程中的经验和教训（如某些参数对结果影响特别大，某些参数组合会产生意外效果等）

**写作风格**：必须是段落式的专业论述，不能简单分点罗列，要像科研论文那样有逻辑地叙述实验过程。
